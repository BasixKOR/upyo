@upyo/core
==========

Shared types and interfaces for Upyo, a simple and cross-runtime library for
sending email messages.


Installation
------------

~~~~ sh
npm  add     @upyo/core
pnpm add     @upyo/core
yarn add     @upyo/core
deno add jsr:@upyo/core
bun  add     @upyo/core
~~~~
